Intruder爆破密码

  1. 先通过proxy抓包发送数据
  2. Postitions
    • 根据业务选择Attack type
      - 设置变量
  3. Payloads
    • Payload set 选择变量
    • Payload type 设置类型（常用Runtime file 写入字典）
  4. Options
    • Grep - Match （可选）设置字符串匹配验证是否爆破成功
  5. Start attack 开始爆破
    • 可以通过 Grep - Match 设置的字符串匹配查看结果
    • 或者通过Length 查看返回长度区分是否成功
